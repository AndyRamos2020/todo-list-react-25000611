To Do List - React Bootstrap
Descripción
Aplicación web tipo To Do List para gestionar tareas y metas personales. Permite visualizar tareas con su respectiva fecha límite.

Nota: Esta versión corresponde únicamente a la maqueta (estructura visual), sin funcionalidad dinámica.

Tecnologías utilizadas
React
React Bootstrap
CSS
Funcionalidades (maqueta)
Campo para ingresar una tarea
Campo para seleccionar fecha
Botón para agregar tarea (visual)
Lista de tareas (estática)
Botón eliminar (visual)
Diseño Responsive
La aplicación se adapta a diferentes tamaños de pantalla gracias a React Bootstrap.

Cómo ejecutar el proyecto
Clonar el repositorio:
git clone https://github.com/AndyRamos2020/todo-list-react-25000611.git
Instalar dependencias:
npm install
Ejecutar el proyecto:
npm run dev
